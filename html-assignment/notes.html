5 HTML tags I have learned in Web Development

  1. Blockcoute: The blockquote HTML element indicates that the enclosed text is an extended quotation.
  2. Cite: The cite HTML element is used to describe a reference to a cited creative work and must include the title
  3. Hr: The hr HTML element represents a thematic break between paragraph
  4. Img:The img HTML element embeds an image into the document.
  5. Heading:The h1 to h6 HTML elements represents six levels of headings, h1 being the highest section and h6 the lowest
